Only a handful of keyboard shortcuts are implemented. Standard IDE shortcuts are missing.

## Implemented

- Ctrl+B: Toggle file explorer
- Ctrl+Shift+B: Toggle editor panel
- Ctrl+Tab: Cycle workspace tabs
- Ctrl+O: Open workspace
- Ctrl+S: Save file (in file windows)

## Needed

- Ctrl+N: New agent window
- Ctrl+W: Close active window
- Ctrl+`: New terminal
- Ctrl+Shift+P: Command palette (see command-palette spec)
- Ctrl+Shift+N: New workspace
- Ctrl+1/2/3...: Focus window by index
- Escape: Deselect / close modals
- Canvas navigation: Home to reset view, +/- to zoom
